## Projects
### Loan Status Predictive Analysis
**Goal**: Predict whether the loan will be approved or defaulted using the features in the dataset provided

**Description**: The dataset includes features such as person's age, education, income, loan amount, loan interest rate, and credit score. This project includes the following steps to analyse and observe insights: Data Loading, Data Cleaning and Preprocessing, perform EDA, create visualization, train and evaluate multiple machine learning models, create Data Lakehouse and develop visual report.

**Tools**: Python (Pandas, Matplotlib, Seaborn, Scikit-learn), PySpark, Microsoft Fabric

### Online Sales Analytics Report
**Goal**: Provide strategic recommendations for the company based on data-driven insights

**Description**: The dataset includes sales data, product information, and invoice details from an international e-commerce company stored in an Azure SQL database. This project includes the following steps: Data Loading, Data Cleaning and Transformation, Data Model Creation, Data visualization, StoryTelling and Reporting.

**Tools**: Power BI, Azure, Power Query


### Gaming Analytics Project
**Goal**: Deliver actionable insights, demonstrating how data analysis can influence decision-making in the gaming industry

**Description**: There are three datasets which include multiple key metrics in a game company. This project includes the following steps to analyse in tasks: Data Loading, Data Cleaning, Data Analysis and join tables with MySQL, perform window functions, Reporting and insights.

**Tools**: MySQL


### Housing Price Predictive Analysis
**Goal**: Predict the price of a house based on various features by creating multiple Model

**Description**: The dataset contains various features such as Square Feet, Number of Bedrooms. This project includes the following steps to analyse in tasks: Data Loading, Data Cleaning, perform EDA, create visulalisation and correlations, create a muliple linear regression model and random forest model, compare the RMSE of each model, conclude the best model.

**Tools**: Python (Pandas, Matplotlib, Seaborn, Scikit-learn), Microsoft Fabric
